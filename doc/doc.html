Softwarové požadavky

Doporučené je používat Python 3. Flask ale podporuje Python 3.5 a novější, Python 2.7, and PyPy.

Databáze

Je třeba mít nainstalovaný MySQL server. Dále vytvořenou databázi s názvem Transport.

Instalace

  1. Zkopírujte .env.example do .myenv
  2. V .myenv nahraďte obsah DATABASE_URI vaším připojením k databázi
    DATABASE_URI=mysql+pymysql://root:@localhost/transport
  3. Spusťte python virtuální prostředí
  4. Vyexportujte proměnné prostředí
  5. Aktivujte virtuální prostředí
  6. Nainstalujte závislosti
  7. Spusťte aplikaci
  1. cp .env.example .myenv
  2. DATABASE_URI=mysql+pymysql://root:@localhost/transport
  3. virtualenv venv
  4. set -a
    . .myenv
    set +a
  5. source venv/bin/activate
  6. pip install -r requirements.txt
  7. flask run --host=0.0.0.0
